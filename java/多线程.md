# 多线程

## 线程创建
    1. new Thread(runable)
    
    2. class MyThread extends Thread {
        @Override
        public void run() {
            // do something
        }
    }

    推荐应方法1，可以很方便的过渡到线程池。

## 线程中断
    interrupt, isInterrupted

> 如果每次工作迭代都调用sleep, isInterrupted判断将没有必要，也没有用处。因为如果isInterrupted被设置，
sleep调用将清除这个状态，并抛出InterruptException
